## Logo Sources:

You can download logos from:
- Official insurance company websites
- https://logos-world.net/ (search for each company)
- Company press kits or media resources

## Image Format:

- Preferred format: PNG with transparent background
- Recommended size: 200-300px width
- File naming: Use lowercase with hyphens (as listed above)
